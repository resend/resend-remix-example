# Resend with Remix

This example shows how to use Resend with [Remix](https://remix.run).

## Instructions

1. Define environment variables in `.env` file.

2. Install dependencies:

  ```sh
npm install
# or
yarn
  ```

3. Run Remix locally:

  ```sh
npm run dev
  ```

4. Open URL in the browser:

  ```
http://localhost:3000/send
  ```
